# Retail Sales SQL Analysis

## Project Overview
This project analyzes a retail dataset using SQL to identify revenue trends and top selling products.

## Dataset
Retail sales dataset containing:
- product
- region
- price
- quantity
- order date

## Analysis Performed
- Total revenue calculation
- Top selling products
- Regional revenue analysis
- Monthly sales trend

## Skills Demonstrated
SQL
Data Analysis
Aggregation
Business Insights

## Key Insights

- Laptop generates the highest revenue.
- South region shows strong sales performance.
- Monthly revenue increases towards later months.
