# Retail Sales SQL Analysis

## Project Overview
This project analyzes a retail dataset using SQL to identify revenue trends and top selling products.

## Dataset
Retail sales dataset containing:
- product
- region
- price
- quantity
- order date

## Analysis Performed
- Total revenue calculation
- Top selling products
- Regional revenue analysis
- Monthly sales trend

## Skills Demonstrated
SQL
Data Analysis
Aggregation
Business Insights

## Business Questions

1. What is the total revenue generated?
2. Which products sell the most?
3. Which region contributes the highest revenue?
4. How does revenue change month by month?

## SQL Techniques Used

- SUM()
- GROUP BY
- ORDER BY
- Aggregation

## Key Insights

- Laptop generates the highest revenue due to higher price.
- South region contributes the largest share of sales.
- Revenue increases gradually in later months.
